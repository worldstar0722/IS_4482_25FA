# TASK II : Changing existing code blocks

import matplotlib.pyplot as plt
import numpy as np

#range of data
x=np.arange(100)
y=np.random.normal(200,2,100)

#Create plots
fig, ax=plt.subplots()
ax.plot(x,y)

#Color,Transparency edit, alpha value
ymin=ax.get_ylim()[0]
ax.fill_between(x, y, ymin, facecolor='y', alpha=.45)

#Title, label
plt.title("Cost over time")
plt.xlabel("time")
plt.ylabel("cost")

#display.display(fig)여
plt.show()

I modified the graph in the following ways:

  • I changed the background color of the area under the line to yellow using facecolor='y'.
  • I set the transparency using alpha=0.45, which controls how see-through the background color appears. Lower alpha means more transparent, and higher alpha makes it more solid.
  • I updated the graph title to "Cost over time".
  • I added axis labels: "time" for the x-axis and "cost" for the y-axis to make the graph easier to interpret.

Axis labels are important because they show what each part of the graph represents. The alpha parameter is useful for visual clarity — it helps you balance visibility when overlaying elements. When I tried changing the alpha value, the yellow area became lighter or darker depending on the number.
